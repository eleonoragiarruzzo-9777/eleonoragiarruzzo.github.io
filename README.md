## Il progetto

Questo portfolio è un sito statico costruito da zero in **HTML, CSS e JavaScript vanilla** — nessun framework, nessun template, nessuna scorciatoia.

La scelta non è nostalgica: è intenzionale. Costruire senza astrazione costringe a capire ogni singola riga. Ed è esattamente il tipo di apprendimento che mi interessa.

### Stack

```
index.html   →  struttura e markup semantico
style.css    →  layout, design system, animazioni
main.js      →  interazioni (Intersection Observer, smooth scroll)
```

### Caratteristiche tecniche

- Layout **fully responsive** (mobile-first con breakpoint a 900px e 600px)
- **CSS custom properties** per un design system coerente e modificabile
- Animazioni tramite **CSS keyframes** e **Intersection Observer API** — zero librerie
- Font: **Syne** (display) + **DM Mono** (body) — caricati da Google Fonts
- Tema dark con palette neutra e accento verde acido (`#c8f55a`)
- Nessun cookie, nessun tracker, nessuna dipendenza esterna oltre ai font

## Come usarlo in locale

Nessuna installazione richiesta.

```bash
# Clona o scarica i file
git clone https://github.com/eleonora/portfolio.git

# Apri nel browser
open index.html
# oppure
npx serve .
```

I font richiedono connessione internet. Tutto il resto funziona offline.
